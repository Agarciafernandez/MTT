The GOSPA code in this folder was written by Abu Sajana Rahmathullah 

https://github.com/abusajana/GOSPA

The code also implements an improvement suggested by Jan Krejčí in February 2023 using the assign2D function in the TrackerComponentLibrary instead of the auction algorithm

D. F. Crouse, "The tracker component library: free routines for rapid prototyping," in IEEE Aerospace and Electronic Systems Magazine, vol. 32, no. 5, pp. 18-27, May 2017

https://github.com/USNavalResearchLaboratory/TrackerComponentLibrary

The folder also includes the compiled version of assign2D for Windows systems.

The folder also includes an implementation of the GOSPA quasi-metric to enable different penalties for missed and false targets [1].

[1] A. F. García-Fernández, J. Gu, L. Svensson, Y. Xia, J. Krejčí, O. Kost, O. Straka “GOSPA and T-GOSPA quasi-metrics for evaluation of multi-object tracking algorithms,” accepted in IEEE Transactions on Aerospace and Electronic Systems, 2026.
